# Security Policy

## Supported Versions
IncidentDesk is currently in MVP stage. Security fixes are applied on the latest `main` branch.

## Reporting a Vulnerability
Please report vulnerabilities responsibly:
- Open a private advisory in GitHub Security tab when possible.
- If private advisory is not available, contact the maintainer directly before public disclosure.

Include:
- clear reproduction steps
- impact assessment
- affected components/routes
- suggested remediation if available

## Scope Notes
IncidentDesk is defensive DFIR software. Contributions that add offensive capabilities are out of scope.
